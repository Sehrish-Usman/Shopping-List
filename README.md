# 🛍️ Shopping App (Flexbox Based)

A responsive and beginner-friendly **Shopping Web App** built using **HTML, CSS (Flexbox), and JavaScript**.
This project demonstrates core front-end concepts like DOM manipulation, event handling, and dynamic UI updates.

---

## 🚀 Features

### 🛒 Product Section

* Display products (Class, Lipstick, Bangles, Shoes, Abaya, Scarf)
* Each product has:

  * Image
  * Name
  * Price
* "Add to Cart" functionality

### 🧺 Cart System

* Add items to cart
* Increase / Decrease quantity (+ / -)
* Remove items from cart
* Auto total price calculation

### 📝 Manual Item List

* Input field to add custom items
* Add button to insert item into list
* Delete button for each item
* Live total items counter

### 📱 Responsive Design

* Built using **Flexbox**
* Works on:

  * Mobile 📱
  * Tablet 📲
  * Desktop 💻
